Goal

Analyze level difficulties by applying discrete mathematics concepts.


Contributions

    - Calculated #possible final board configurations

    - Implemented game in Unity

Analysis: Total #possible final board configurations

For each combination of horizontal/vertical cars (ex: 3 vertical cars, 1 horizontal and 1 vertical, etc.), counted the total number of all arrangements of the board for when the solution was found.

More possible arrangements indicated a more difficult level, since there was more room for error.



Game Implementation

The actual Rush Hour game was created using Unity.



Features

    - 2x1 horizontal and vertical cars

    - Snap-to-place cars

    - #Moves counter

    - Multiple levels of increasing difficulty
